## 1. 見出し:

```markdown
# 大見出し (Heading 1)
## 小見出し (Heading 2)
### さらに小さい見出し (Heading 3)
```

## 2. テキストの装飾:

```markdown
*斜体* もしくは _斜体_
**太字** もしくは __太字__
~~取り消し線~~
```

## 3. リスト:

```markdown
- 番号なしリスト
- 項目1
- 項目2

1. 番号付きリスト
2. 項目1
3. 項目2
```

## 4. 引用:

```markdown
> これは引用文です。
```

## 5. リンク:

```markdown
[リンクテキスト](https://www.example.com)
```

## 6. イメージ:

```markdown
![代替テキスト](https://www.example.com/image.jpg)
```

## 7. 水平線:

```markdown
---
```

## 8. コード表示:

```markdown
インラインコード: `code`
```

```markdown
コードブロック:
```
```python
def hello():
    print("Hello, Markdown!")
```

## 9. テーブル:

```markdown
| 列1 | 列2 |
|-----|-----|
| 1   | A   |
| 2   | B   |
```

## 10. 脚注:

```markdown
テキストに脚注を追加します。[^1]
[^1]: 脚注の内容はここに書きます。
```

## 11. 自動リンク:

```markdown
<https://www.example.com>
<email@example.com>
```

## 12. エスケープ:

```markdown
特殊文字をエスケープする場合は \* などを使用します。
```

以上が、全ての内容をマークダウン形式で書いたものです。これらのマークダウンを使って、自由に記事を書いてみてください！
